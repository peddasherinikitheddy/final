Accessible(?) and responsive tabs (tabgroups). Html and Css only

Themed: theme-plum

To do:

  • Verify accessibility. Problems / solutions?
  • Verify browser compatibility. Currently tested on:
    • Chrome 34 (Linux), Chrome 36 (Android/Windows).
    • Firefox 27, 29, 31 (Windows / Linux).
    • Internet Explorer 11.
    • Opera 12 (Windows).

Features:

  • No JavaScript required.
  • Flexible. Stretches with the surrounding space and the tab content.
  • Can handle multiple tabgroups and nested tabgroups.
  • Retains correct tab order, between tabs and content, as well as the surrounding.
  • Tabs can be navigated by using the arrow keys:
    • left / up - moves left / up.
    • right / down - moves right / down.
  • Responsive. Switches between standard/left tabs and the stacked full width tabs, at a specified viewport width.
  • If there are many tabs or very long tab captions, in large screen mode the standard tabs break off in multiple rows if there is not enough of space.

Usage example

<div class="ui-tabgroup">
  <input class="ui-tab1" type="radio" id="tgroup_a_tab1" name="tgroup_a" checked />
  <input class="ui-tab2" type="radio" id="tgroup_a_tab2" name="tgroup_a" />
  <input class="ui-tab3" type="radio" id="tgroup_a_tab3" name="tgroup_a" />
  <div class="ui-tabs">
    <label class="ui-tab1" for="tgroup_a_tab1">Caption 1</label>
    <label class="ui-tab2" for="tgroup_a_tab2">Caption 2</label>
    <label class="ui-tab3" for="tgroup_a_tab3">Caption 3</label>
  </div>
  <div class="ui-panels">
    <div class="ui-tab1">
      <h3>Caption 1</h3>
      <p>Lorem ipsum dolor sit amet.</p>
    </div>
    <div class="ui-tab2">
      <h3>Caption 2</h3>
      <p>Ut enim ad minim veniam.</p>
    </div>
    <div class="ui-tab3">
      <h3>Caption 3</h3>
      <p>Duis aute irure dolor.</p>
    </div>
  </div>
</div>

The radio buttons require a name for keeping the individual buttons together as a group. To use multiple tabgroups, the radio button name has to be unique per each tabgroup.

Each radio button requires a unique id, that each corresponding label can connect to via the "for" attribute.

Each radio button, label and panel require a numbered class "ui-tabX", where X can be 1 to 9. (The maximum of 9 can easily be extended.)

Icons only

For use in combination with left-tabs. Shrinks the tabs down to a width 50px. And due to the small space the tabs take, they are no longer adapted to smaller viewports.

(Currently only available in the Plum theme.)

Just add the class "icons-only" like so:

<div class="ui-tabgroup left-side icons-only">
